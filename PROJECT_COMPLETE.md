## 💡 CUSTOMIZATION

### **Change Colors**
File: `frontend/css/styles.css`
```css
:root {
    --primary-color: #00a86b;     /* Green */
    --secondary-color: #ff6b35;    /* Orange */
}
```

### **Update Pricing**
File: `frontend/js/script.js`
```javascript
const pricing = {
    '06:00 - 07:00': 800,  // Change prices here
    ...
};
```

### **Edit Text**
File: `frontend/index.html`
- All text is directly editable
- Search for text you want to change
- Update and save

### **Add Images**
1. Put images in: `frontend/assets/`
