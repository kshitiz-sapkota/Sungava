## 💡 CUSTOMIZATION TIPS

### Change Prices:
**File:** script.js (line 28)
```javascript
const pricing = {
    '06:00 - 07:00': 800,  // Your price
    '12:00 - 13:00': 1000, // Your price
};
```

### Change Colors:
**File:** styles.css (line 10)
```css
:root {
    --primary-color: #00a86b;  /* Your color */
    --secondary-color: #ff6b35; /* Your color */
}
```

### Change Location:
**File:** index.html (line ~280)
```html
<p>Kathmandu, Nepal</p>  <!-- Your location -->
```

### Add/Remove Time Slots:
**File:** script.js (line 28)
```javascript
// Add more time slots to pricing object
'22:00 - 23:00': 1500,  // Late night slot
```
