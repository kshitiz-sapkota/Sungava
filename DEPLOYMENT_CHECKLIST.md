## 🚨 COMMON ISSUES & FIXES

### Issue: Website shows 404 Error
**Fix:** 
- Ensure index.html is in root directory (public_html)
- Check file name is exactly "index.html" (lowercase)

### Issue: WhatsApp doesn't open
**Fix:**
- Verify phone number format: 9779801234567
- No spaces, no special characters
- Includes country code (977)

### Issue: Styling looks broken
**Fix:**
- Clear browser cache (Ctrl + F5)
- Verify styles.css is uploaded
- Check file path in index.html

### Issue: Booking system doesn't work
**Fix:**
- Verify script.js is uploaded
- Check browser console for errors (F12)
- Ensure JavaScript is enabled in browser

### Issue: Mobile menu doesn't open
**Fix:**
- Clear cache and reload
- Test in different browser
- Check script.js is loaded

### Issue: Past dates can be selected
**Fix:**
- This is normal if testing with future date
- System automatically disables past dates
