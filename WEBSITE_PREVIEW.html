🎯 How to Use Your Website

Option 1: Simple Way (No Setup Required)

  1. Navigate to: C:\Royal\frontend\
  2. Double-click on: index.html
  3. Your website will open in your default browser!

Option 2: With Live Server (Recommended)

  1. Open VS Code
  2. Install "Live Server" extension
  3. Right-click on index.html
  4. Select "Open with Live Server"
  5. Auto-refreshes when you make changes!

🔧 Backend Setup (Optional - For Advanced Features)

Note: The frontend works perfectly without the backend. Use backend for database storage and advanced features.

  1. Install Node.js from nodejs.org
  2. Open Command Prompt
  3. Navigate to backend: cd C:\Royal\backend
  4. Install dependencies: npm install
  5. Copy config: copy .env.example .env
  6. Start server: npm start
