⚡ Quick Setup (3 Steps)

Update Contact Information

Open index.html in any text editor and find/replace __________ with:

  • WhatsApp: Your number with country code (e.g., 9779801234567)
  • Phone: Your phone number
  • Facebook: Your Facebook page name

Tip: Use Ctrl+H (Windows) or Cmd+F (Mac) to find and replace all at once!

Upload to Web Hosting

Upload these 3 files to your hosting public_html folder:

  • index.html
  • styles.css
  • script.js

Use cPanel File Manager or FTP client like FileZilla

Test & Launch

Open your website and test:

  • ✓ Select a date from calendar
  • ✓ Choose a time slot
  • ✓ Fill booking form
  • ✓ Verify WhatsApp opens
  • ✓ Test on mobile device
